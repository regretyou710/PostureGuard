# 📱 PostureGuard (姿勢守護者)

**告別低頭族！結合實時硬體感測與動態視覺回饋，專注守護您的頸椎健康。**

## 📖 專案概念

現代人因長時間低頭使用手機，導致「頸椎症候群 (Text Neck)」成為普遍的文明病。
PostureGuard 是一款專為解決此痛點而生的 React Native 應用程式。本專案捨棄了傳統健康 App 繁瑣的數據報表，採用「專注陪伴模式」理念：當使用者在辦公、閱讀或觀看影片時，只需開啟此 App 並將手機放置於桌面支架或雙手握持，它便會化身為您專屬的「姿態健康檢視工具」。

搭配 `expo-keep-awake` 保持螢幕常亮，程式會透過手機內建的加速計，無延遲捕捉設備的傾斜角度。本專案採用流暢的 Lottie 動態圖示與觸覺回饋，給予最直覺的即時狀態指示：

* ✅ **健康角度 (抬頭挺胸)：** 畫面呈現舒緩的綠色與正面動態圖示，代表姿態良好。
* ⚠️ **警告角度 (輕微偏移)：** 當設備角度過度平放，畫面無縫過渡至警告橘色，提醒您注意頸部壓力。
* 🚨 **危險角度 (嚴重低頭/後仰)：** 當角度進入危險區間，畫面將切換為警示紅色，並伴隨**強烈的實體震動**，直到使用者將手機角度修正。

這不僅是一個實用的防低頭工具，更是展示 React Native 呼叫設備底層硬體、處理防抖邏輯與狀態機管理的高效能實踐。

## ✨ 核心功能

* **即時雙向傾角偵測：** 無延遲讀取手機三軸數據，完美覆蓋「往前低頭」與「往後躺仰」的雙向不良姿勢盲區。
* **三階段狀態回饋：** 結合視覺 (Lottie 狀態切換、背景色漸變) 與觸覺 (Haptic Feedback) 的多重感官提醒。
* **專注閱讀模式：** 內建螢幕常亮機制，確保在姿態良好的情況下，螢幕不會因閒置而鎖定中斷體驗。
* **防抖與效能優化：** 實作時間延遲 (Debounce) 機制，並優化感測器採樣頻率，過濾手部微小晃動引發的雜訊與誤判。

## 🛠️ 技術亮點

本專案在技術實作上具備以下三個核心亮點，展現對前端架構與原生硬體特性的掌握：

### 1. 空間數學運算與「雙向盲區」突破

從感測器取得的原始數據是 Y, Z 軸的重力分量，專案中實作了空間幾何轉換邏輯 `Math.abs(Math.atan2(y, z) * (180 / Math.PI))` 將其轉換為直覺的俯仰角。
**實作細節：** 為了避免只偵測單向低頭的盲區，算法進一步演進為「**計算與 90 度完美垂直的絕對偏差值**」，僅用極簡的數學邏輯，便完美覆蓋了使用者前傾與過度後仰的各種 Edge Cases。

### 2. 防抖機制與狀態機設計

硬體感測器的高頻數據容易帶有雜訊。本專案引入 `Zustand` 建立全局狀態機，並在 `useEffect` 中實作了 Timeout 防抖機制。
**實作細節：** 角度進入危險區域時不會立刻報警，而是必須「持續維持不良姿勢超過 2 秒」才會觸發狀態改變與震動。這大幅提升了真實世界的使用者體驗，防止「手滑」造成的誤判。

### 3. 資源與生命週期管理

嚴格控管硬體資源。不僅將感測器的更新頻率限制在合理的 500ms 以節省電池效能，更確保在 Component Unmount 或狀態重置時，精準執行 `subscription.remove()` 與 `clearTimeout`，徹底杜絕 Memory Leak 的風險。

## 💻 技術棧

* **框架:** React Native (Expo)
* **硬體感測:** `expo-sensors` (Accelerometer)
* **震動回饋:** `expo-haptics`
* **動畫引擎:** `lottie-react-native`
* **狀態管理:** `Zustand`
* **設備控制:** `expo-keep-awake`
